# blockchain
Block chain test

# Dependency
* python3.6
* Flask 0.12.2
* requests 2.18.4
1. Set up environment
```
conda create -n py36 python=3.6 anaconda Flask==0.12.2 requests==2.18.4
source acrivate py36
```
2. Run the server
python blockchain.py 5000
3. Test
python -m unittest blockchain.py
